# Flag Checker для GitHub Pages

Это полностью статическая версия сайта:
- загружает `.docx` в браузере,
- сверяет текст со справочником `data/registries.json`,
- подсвечивает совпадения в предпросмотре,
- обновляет справочник через GitHub Actions.

## Что важно
- Это версия без сервера. Она не переписывает исходный `.docx`.
- Для первого запуска достаточно загрузить все файлы в корень репозитория.
- Для обновления списков зайдите в Actions → update-registries → Run workflow.
